
# Quiz App Kotlin

a simple quiz android Kotlin App that contain 
three screens Start , Questions and Result screen

- Android Kotlin
- MVVM Pattern
- Navigation Component
- Retrofit Library
- Room Database
- Json

## Roadmap

- clone the App

- run it on Android Studio
- Make sure that there is a network connection when run the App
